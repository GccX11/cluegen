Codenames Clue Generator

How to use
×

Instructions on how to use the Codenames Clue Generator:

  1. Use the "Generate Clues" button to generate a set of clues.
  2. Use the "Edit Text" button to modify the generated clues.
  3. Use the "Change Player" button to cycle between the three teams.
  4. Tap on a word to change the team it belongs to.
  5. Long press on a word to toggle marking it as guessed to remove it from the game.
